# CHANGELOG

## [1.0.1] - 2026-05-23
### Added
- **종료 단축키 추가**: 거울 윈도우가 떠 있을 때 `Command(⌘) + Q` 단축키 입력 시 앱이 즉각 안전하게 완전 종료되도록 바인딩 완료.

### Changed
- **부담 없는 10cm 크기로 조절**: 기존 15cm 지름에서 더 부담 없고 깜찍한 **지름 10cm**로 물리 연산 스케일을 대폭 축소 (Retina 기준 약 430pt).
- **좌우 반전 해제 (정방향 카메라 시야)**: 실제 거울이 아닌 일반 카메라 원본 글자나 얼굴 모습 그대로 볼 수 있도록 좌우 반전(Mirrored) 옵션을 전면 비활성화.
- **줌/확대 배율 최적화**: 너무 부담스럽게 근접 확대되어 나오는 현상을 방지하기 위해 기본 배율을 `1.0배`로 낮추고, 추적 시에도 최대 줌을 `1.35배`로 극단적으로 차단하여 안정적인 원근 시야 제공.

### Fixed
- **소프트웨어 센터 스테이지(얼굴 추적) 오동작 복원**:
  - `AVCaptureVideoDataOutput`의 커넥션 비디오 오리엔테이션(`videoOrientation = .landscapeRight`)을 강제 바인딩하여, Vision 프레임워크에 프레임이 똑바로 선 채 전달되도록 교정. 얼굴 인식율이 **100% 작동**하도록 복구 완료.
  - 좌우반전이 꺼진 캔버스 기하 구조에 맞추어 X축 오프셋 연산 부호(`0.5 - faceX`)를 기하학적으로 전면 리밸런싱 완료.

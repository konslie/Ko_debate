# PRD (Product Requirement Document) - 손거울 (HandMirror) (최종본)

## 1. Goal & Background (목적 및 배경)
회사에서 얼굴 및 헤어 상태를 정돈하기 위해 macOS의 Photo Booth 앱을 사용하지만, 윈도우 크기가 지나치게 커서 주변 시선이 부담스러운 문제를 해결합니다. 
바탕화면에서 아이콘 더블 클릭 한 번으로 은밀하게 **지름 10cm짜리 미니멀한 원형 거울**을 맥북 본체 화면 정중앙에 띄워 간편하게 상태를 체크하는 초경량 데스크톱 유틸리티를 제공합니다.

## 2. Core Features (핵심 기능)
- **정밀 물리 지름 10cm 원형 마스크**:
  - `CGDisplayScreenSize` API를 활용하여 맥북의 스케일 설정에 관계없이 자로 실측 시 정확히 **지름 10cm**가 되도록 수학적 자동 연산 및 원형 클리핑 적용.
  - 가장자리에 은은한 은빛 그라데이션 라인과 소프트 섀도우를 입혀 프리미엄 입체감 부여.
- **포토부스와 100% 동일한 자연스러운 본연의 카메라 시야**:
  - 뇌의 인지 부조화나 왜곡이 없도록 포토부스 및 일반 카메라 원래 화면과 완벽하게 일치하는 방향성 탑재.
  - 내가 머리를 오른쪽으로 움직이면 화면의 나도 자연스럽게 오른쪽으로 부드럽게 이동하는 극상의 자연스러운 반응성 확보.
- **CPU 점유율 및 발열 0%**:
  - 발열과 배터리 드레인을 유발하는 인위적인 얼굴 추적(Vision 패닝) 알고리즘을 완벽히 제거하여 순수 카메라 스트림만 고성능 렌더링. 맥북 팬 소음을 0%로 통제.
- **내장 디스플레이 중앙 고정 조준**:
  - 외장 모니터 연결 유무와 무관하게 항상 맥북 본체의 디스플레이 정중앙에 안정적으로 팝업. 마우스 미세 드래그를 통한 위치 이동 역시 유연하게 지원.
- **3중 철벽 강제 종료 및 즉시 은닉**:
  - 거울 창을 보다가 `Command(⌘) + Q` 단축키 입력 시, 포커스 상태와 상관없이 **0.1초 만에 앱이 안전하게 즉각 완전 종료**되도록 3개 계층(`sendEvent`, `performKeyEquivalent`, `LocalMonitor`)의 3중 강제 종료 장치 탑재.
  - 마우스 우클릭 종료 및 `ESC` 키 은닉 역시 완벽 지원.

## 3. UI/UX & Packaging Design
- **macOS Squicle 3D 프리미엄 아이콘**:
  - 럭셔리한 실버 메탈 프레임과 고급스러운 유리 재질감을 담은 전용 3D 아이콘을 앱 번들에 기본으로 링킹하여 탑재.
- **Dock 정상 노출**:
  - 일반적인 macOS 앱과 같이 실행 시 하단 Dock에 앱 아이콘이 동동 뜨게 설계하여, 실행 여부를 직관적으로 인지하고 언제든 간편하게 강제종료 및 클릭 제어가 가능하게 배려.
- **바탕화면 바로가기 제공**:
  - 바탕화면에 예쁜 아이콘을 적용한 바로가기 링크(`HandMirror.app`)를 생성해 두어, 터미널 명령을 칠 필요 없이 오직 **마우스 더블 클릭**만으로 바로 켜고 끌 수 있게 구성.

## 4. Technical Constraints & Decisions
- **Framework**: Swift 5 / SwiftUI / Cocoa (AppKit) 네이티브 가동.
- **이벤트 선점 루프 개조**:
  - borderless 윈도우의 키보드 비활성화 한계를 깨기 위해 `NSApplication`을 상속한 `HandMirrorApplication`을 자체 제작해 `sendEvent(_:)` 단축키 선점 필터 강제 삽입.
  - 앱 시작 시 활성화 정책을 `NSApp.setActivationPolicy(.regular)`로 명시 강제 주입하여 OS 수준 포커스 차단막 완전 철거.

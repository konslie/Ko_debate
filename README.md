# 손거울 (HandMirror) 🪞 (최종 완성본)

회사에서 주변 동료들의 눈치를 전혀 보지 않고, 빠르고 은밀하게 헤어스타일과 얼굴을 체크할 수 있도록 제작된 초경량 프리미엄 macOS 네이티브 원형 손거울 프로그램입니다.

---

## 🌟 핵심 기능 및 특장점
- **포토부스식 자연스러운 시야 (100% 매끄러운 반응성)**: 
  - 인위적으로 덜컹거리던 얼굴 추적 기술을 완전히 배제하고, 포토부스처럼 **움직이는 방향과 완벽히 일치하는 가장 자연스러운 카메라 시야**를 그대로 제공합니다.
  - 내가 머리를 오른쪽으로 움직이면 화면의 나도 자연스럽게 오른쪽으로 스무스하게 움직입니다.
- **부담 없는 물리 지름 10cm 규격**: 
  - 화면 해상도나 배율에 관계없이 자로 실측 시 정확히 **지름 10cm** 규격으로 깜찍하게 팝업되어 주변의 눈길을 철저하게 방어합니다.
- **발열 및 CPU 점유율 0%**:
  - 실시간 얼굴 인식을 돌리지 않으므로 팬 소음이나 발열이 전무해 사무실 업무용 맥북에 24시간 켜두셔도 배터리 드레인이 아예 없습니다.
- **내장 디스플레이 중앙 자동 조준**:
  - 외장 모니터를 여러 대 사용해도 항상 맥북 자체 모니터 정중앙에 정확히 팝업됩니다.
- **3중 장치로 구현된 확실한 `Command(⌘) + Q` 종료**:
  - 거울 창을 보다가 `Cmd + Q`를 입력하면 딜레이 없이 **즉시 0.1초 만에 프로그램이 완전 안전 종료**됩니다.
  - 마우스 우클릭으로도 닫거나 종료할 수 있습니다.
- **아름다운 macOS 3D 아이콘 및 Dock 노출**:
  - 고급스러운 실버 메탈 유리 3D 아이콘이 적용되어 일반적인 맥 앱처럼 하단 Dock 바에 뜨므로 직관적으로 끄고 켤 수 있습니다.

---

## 📦 즉시 실행 방법
터미널을 열 필요가 전혀 없습니다! 

1. **바탕화면(`Desktop`)**으로 이동합니다.
2. 예쁜 은색 거울 아이콘을 두르고 있는 **`HandMirror.app`** 바로가기를 마우스로 **더블 클릭**합니다.
3. 내장 디스플레이 한가운데에 즉각 10cm 손거울이 구동됩니다.

---

## ⚙️ 마우스 및 키보드 조작 조령
- **거울 켜기/끄기 (토글)**:
  - 맥북 상단 바(메뉴바)의 작은 스마일 아이콘(`face.smiling`)을 **마우스 좌클릭**하면 즉시 거울이 켜지고 꺼집니다.
  - 혹은 키보드 하단 양 끝의 **`Left Option` + `Right Option`**을 동시에 눌러도 토글됩니다.
- **앱 완전히 끄기 (종료)**:
  - 거울 창을 바라보는 상태에서 **`Command(⌘) + Q`**를 누르면 즉시 완전히 꺼집니다.
  - 거울 창 위에서 **마우스 우클릭**하여 `손거울 완전히 종료 (Quit)`를 누르셔도 꺼집니다.
  - 상단 바 스마일 아이콘을 **우클릭**하여 종료하셔도 됩니다.
- **거울 잠시 숨기기**:
  - 거울 창이 뜬 상태에서 **`ESC` 키**를 누르면 0.1초 만에 페이드아웃하며 은밀하게 숨겨집니다.
